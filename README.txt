To use the files format.py
Installing python to your computer
Type the command: python <file_name>
The file format.py will replace the missing values with "NaN" and outputs 3 files data-1.txt, data-2.txt, data-3.txt corresponds to 3 data sets.

To use the files mode.py, mean.py:
Type the command: python <file_name>
These two files will generate files with completed data.

To run the files knn.py, svd.py:
Go inside fancyimpute-0.3.1 folder
Type the command: python <file_name>
These two files will generate files with completed data.
